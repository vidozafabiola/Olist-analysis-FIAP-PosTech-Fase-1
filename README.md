# Análise do Desempenho Comercial – Olist (2017–2018)

Este projeto apresenta uma análise de dados de e-commerce utilizando Python e Power BI, com foco na construção de indicadores de desempenho (KPIs).

## Conteúdo
- Limpeza e preparação de dados
- Construção de datasets analíticos
- Análise com KPIs de desempenho
- Visualização no Power BI

## Ferramentas
- Python (Google Colab)
- Power BI
